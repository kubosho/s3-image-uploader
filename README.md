# S3 image uploader

Image uploader for AWS S3.

## Requirements

- Deno

## Development

Launch development server:

```bash
deno task dev
```

Execute build:

```bash
deno task build
```

Run lint:

```bash
deno task lint
```

Run test runner:

```bash
deno task test
```

Launch storybook:

```bash
deno task storybook
```
